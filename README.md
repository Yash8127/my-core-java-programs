## 📖 About Repository

This repository contains my **Core Java programming practice** completed while learning Java fundamentals and preparing for technical interviews.

The repository includes programs ranging from beginner to advanced level, covering Java syntax, Object-Oriented Programming, Collections Framework, String manipulation, Exception Handling, and common coding interview questions.

My goal is to build a strong Java foundation through consistent hands-on coding.
